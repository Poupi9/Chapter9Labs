## Description of basic commands:

   - `pwd` — Prints the current working directory.
   - `ls` — Lists the files and directories in the current location.
   - `git init` — Initializes a new Git repository in the folder.
   - `git status` — Shows the current state of the repository, including changes to be committed, staged, or untracked files.
   - `git add` — Stages files to be included in the next commit.
   - `git commit` — Saves the staged changes with a message, creating a new snapshot in the repository history.
